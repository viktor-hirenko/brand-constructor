# Figma vs PRD v2 - Discrepancies

## Critical Discrepancies

### 1. Concept Selection Limit
- **Figma**: "О до 2-х концептів" (до 2 концептов)
- **PRD v2**: "ЛИШЕ ОДИН концепт"
- **Resolution**: Следовать PRD - только 1 концепт

### 2. "Зв'язок з іншим продуктом" Field
- **Figma**: Присутствует в Brand Basics
- **PRD v2**: Не упоминается явно
- **Resolution**: Добавить поле, уточнить у PO

### 3. Create New Concept Modal
- **Figma**: Базовая форма (название, описание)
- **PRD v2**: Детальный бриф с референсами, целями, ЦА
- **Resolution**: Реализовать полную форму по PRD

### 4. Marketing Package Timeline
- **Figma**: Не указаны сроки
- **PRD v2**: Конкретные сроки для каждого пакета
- **Resolution**: Добавить сроки в карточки пакетов

### 5. CEO Review Step
- **Figma**: Отсутствует
- **PRD v2**: Описан как отдельный этап
- **Resolution**: Реализовать как Step 10 (Review & Submit)

### 6. Export/Download Step
- **Figma**: Отсутствует
- **PRD v2**: Описан экспорт брифа
- **Resolution**: Добавить в Step 10

## Minor Discrepancies

### 7. External Naming Limit
- **Figma**: Визуально до 3 выборов
- **PRD v2**: "від 1 до 3 неймінгів"
- **Status**: Согласовано

### 8. Internal Naming
- **Figma**: 1 выбор
- **PRD v2**: 1 выбор
- **Status**: Согласовано

### 9. Visual Components Incompatibility
- **Figma**: Warning UI присутствует
- **PRD v2**: Логика несовместимости описана
- **Status**: Согласовано, нужна таблица несовместимостей

### 10. Delegate to Designers
- **Figma**: Toggle + confirmation modal
- **PRD v2**: Описана функция делегирования
- **Status**: Согласовано
