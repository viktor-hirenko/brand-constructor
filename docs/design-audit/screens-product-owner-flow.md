# Product Owner Flow - Screen Descriptions

Figma: https://www.figma.com/design/aCe1cUDNaA7Tmcn9Mmvv1w/Brand-Builder?node-id=0-1&m=dev

## 13 Main Screens

### 1. Landing Page
- Заголовок: "Бриф на створення бренду"
- Кнопка "Почати"
- Описание процесса

### 2. Brand Basics (Step 1)
- GEO selector
- Launch date picker
- "Зв'язок з іншим продуктом" field
- Comment textarea
- iPhone preview справа

### 3. Mode Selection (Step 2)
- Light/Dark карточки выбора
- iPhone preview меняется в зависимости от выбора

### 4. Concept Selection (Step 3)
- Grid карточек концептов (thumbnail + название)
- Checkbox для выбора
- Кнопка "Детальніше" для просмотра
- Кнопка "Створити новий концепт"

### 5. Concept Detail View
- Модалка с полной информацией о концепте
- Visual, Logo, Preview изображения
- Описание
- Связанные неймінги

### 6. Create New Concept Modal
- Форма брифа на новый концепт
- Поля: название, описание, референсы
- Кнопка отправки

### 7. External Naming (Step 4)
- Grid карточек неймингов
- Название + слоган
- Можно выбрать до 3 штук

### 8. Internal Naming (Step 5)
- Grid карточек внутренних неймингов
- Выбор только 1

### 9. Brand Preview (Step 6)
- Сводка всех выборов
- Выбранный концепт
- Выбранные неймінги
- iPhone preview

### 10. Marketing Package (Step 7)
- 6 карточек PR пакетов
- Иконка, название, краткое описание
- Кнопка "Детальніше"
- Модалка с полной информацией

### 11. Deliverables (Step 8)
- Legal Landing toggle
- Partner Landing toggle
- Comment textarea

### 12. Visual Components - Empty (Step 9)
- Аккордеон с типами компонентов
- Пустое состояние до выбора

### 13. Visual Components - Filled (Step 9)
- Выбранные варианты в аккордеоне
- iPhone preview с выбранными компонентами
- Warning о несовместимости (если есть)
- Toggle "Довірити вибір дизайнерам"
