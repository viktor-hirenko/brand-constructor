# Адмін-панель бібліотек Brand Constructor + Cloudflare Workers інфраструктура

> **Джерело**: Jira-задача (оригінальний текст)

---

## Key details

### Description

Відповідно до документа необхідно реалізувати адмін-панель для керування бібліотеками Brand Constructor та підготувати відповідну інфраструктуру на базі Cloudflare Workers. Конструктор розміщується на девелоперському домені та використовує Cloudflare R2 для зберігання асетів.

### У межах задачі потрібно:

1. Створити Cloudflare R2-bucket для зберігання асетів бібліотек конструктора брендів в аккаунті upstars_landings.
2. Реалізувати Cloudflare Worker для роботи з бібліотеками (зберігання та отримання даних, звʼязування сутностей, завантаження асетів).
3. Розгорнути адмін-панель Brand Constructor на девелоперському домені розробників.
4. Реалізувати повноцінні адмін-інтерфейси керування бібліотеками:
   - концепти
   - external naming
   - internal naming
   - PR-пакети
   - UI-компоненти
5. Передбачити можливість звʼязування сутностей між бібліотеками (наприклад, привʼязка external naming до концептів).
6. Реалізувати базову модель ролей доступу для контент-адміністраторів.

### Робота з асетами

Адмін-панель має підтримувати завантаження та зберігання графічних матеріалів бібліотек, що передбачені документом.

Для завантажуваних зображень потрібно:

- перевіряти співвідношення сторін відповідно до типу асета (поки ми не знаємо фінальних співвідношень, треба зробити тестові взявши наявні асети із прототипа і їх співвідношення сторін)
- Зробити підтримку файлів у форматі PNG та за можливості SVG
- відхиляти файли, що не відповідають вимогам
- забезпечити коректне зберігання та доступ до файлів через R2
